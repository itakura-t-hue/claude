# Google AI Studio用プロンプト：圧倒的にドラマティックなUIのWebサイト

以下のプロンプトをGoogle AI Studioにコピペして使ってください。

---

## プロンプト本文

```
あなたはシニアUIデザイナー兼フロントエンドエンジニアです。
以下の仕様に従って、1つの .html ファイルで完結する、圧倒的にドラマティックで余白を活かしたWebサイトを作成してください。

━━━━━━━━━━━━━━━━━━━━
■ 全体方針
━━━━━━━━━━━━━━━━━━━━

【スタイル】
- 映画のオープニングのような "シネマティック" な雰囲気
- ミニマリズム × ダークモードベース
- 余白（ホワイトスペース）を「演出」として意図的に大量に使う
- 各セクションの上下に padding: 120px〜200px を確保し、呼吸感を出す

【カラーパレット】
- 背景: #0a0a0f（ほぼ黒、わずかに紫がかった深い闇）
- テキスト（メイン）: #f5f5f5
- テキスト（サブ）: #888888
- アクセント: #00D9FF（シアンネオン）
- セカンダリアクセント: #FF006E（マゼンタ）
- グラデーション: linear-gradient(135deg, #00D9FF, #7C3AED, #FF006E) を控えめに使用

【タイポグラフィ】
- 見出し: 'Syne' (Google Fonts), Bold 700, letter-spacing: -0.03em
- 本文: 'Epilogue' (Google Fonts), Regular 400, line-height: 1.7
- 数字やラベル: 'Space Mono' (Google Fonts) をアクセントとして混ぜる
- ヒーローの見出しは clamp(3rem, 8vw, 7rem) で大胆に大きく
- Google Fonts読み込み: <link href="https://fonts.googleapis.com/css2?family=Syne:wght@700;800&family=Epilogue:wght@300;400;500&family=Space+Mono:wght@400;700&display=swap" rel="stylesheet">

━━━━━━━━━━━━━━━━━━━━
■ 構成（セクションごとにレイアウトを変えること）
━━━━━━━━━━━━━━━━━━━━

【1. ヒーローセクション】
- 全画面表示（100vh）
- 背景に超淡いグラデーションのオーブ（radial-gradient）を2〜3個配置。ゆっくりアニメーション（20s ease-in-out infinite）
- 中央に1行のキャッチコピー。巨大フォント（clamp(3rem, 8vw, 7rem)）
- キャッチコピーの一部分だけにグラデーションテキスト（background-clip: text）
- その下に1行の説明文（#888, font-size: clamp(1rem, 2vw, 1.25rem)）
- CTAボタン1つ。ボーダーのみ（ghost style）で、hoverでグラデーション背景が現れる
- 最下部にスクロールインジケーター（小さな矢印 or 線がゆっくりバウンス）

【2. フィーチャーセクション】— 左右分割レイアウト
- 3つの特徴を「左テキスト + 右ビジュアル」→「右テキスト + 左ビジュアル」と交互に配置
- ビジュアル側にはCSSだけで作った幾何学的な装飾（グラデーション円、ぼかし、回転する線）
- テキスト側にはラベル（Space Monoで「01」「02」「03」）→ 見出し → 説明文
- 各アイテム間の余白は最低 100px

【3. 数字で見るセクション】— 大きな数字
- 横並びで3〜4つのメトリクス
- 数字は clamp(3rem, 6vw, 5rem) で大きく、Space Mono、グラデーションテキスト
- 下に短いラベル（Epilogue, #888）
- 背景に極めて薄いグリッドパターン（opacity: 0.03）

【4. テスティモニアルセクション】— 全幅テキスト
- 中央に大きな引用文（clamp(1.5rem, 3vw, 2.5rem)）
- 引用符を大きなSVGまたはCSSで装飾的に配置
- 名前と肩書きを下に小さく
- 背景をわずかに明るく（#0f0f18あたり）してセクション区切りを演出

【5. CTAセクション】— 余白の極み
- 上下に200px以上のpadding
- 中央に1行の大きなステートメント（見出しサイズ）
- その下にボタン1つ
- 背景にうっすらとグラデーションオーブ

【6. フッター】
- 極めてシンプル。コピーライトとリンク2〜3本
- 上に細い線（border-top: 1px solid rgba(255,255,255,0.1)）

━━━━━━━━━━━━━━━━━━━━
■ アニメーション・インタラクション
━━━━━━━━━━━━━━━━━━━━

- スクロールアニメーション: IntersectionObserver を使用
  - フィーチャーは左から or 右から fadeIn（交互に方向を変える）
  - 数字セクションはスケールアップ
  - テスティモニアルはフェードイン
  - 全て 0.8s cubic-bezier(0.16, 1, 0.3, 1) のイージング
- ホバー:
  - ボタンは scale(1.02) + グラデーション背景出現
  - リンクは下線がスライドイン（::after疑似要素）
- 背景オーブのアニメーション: translateとscaleをゆっくり繰り返す（15-25秒周期）
- prefers-reduced-motion 対応: アニメーションを全て無効化

━━━━━━━━━━━━━━━━━━━━
■ AIっぽさを排除するための重要ルール
━━━━━━━━━━━━━━━━━━━━

絶対に守ること：

1. 【テキスト】「革新的な」「シームレスな」「包括的な」「最適な」「〜を実現します」のようなAI定型フレーズを使わない。口語的で具体的な表現にする
2. 【レイアウト】3カラム均等グリッドの繰り返しは禁止。セクションごとに構造を変える
3. 【余白】全セクションのpaddingを同じにしない。120px / 160px / 200px のようにリズムを変える
4. 【カードサイズ】もしカードを使うなら、全て同じサイズにしない
5. 【hover効果】全要素が同じhoverパターン（translateY(-4px) + box-shadow）にならないこと
6. 【アニメーションdelay】等差数列（0.1s, 0.2s, 0.3s）にしない。0.1s, 0.15s, 0.3s のように揺らす
7. 【色】背景はベタ塗りの#000ではなく、微妙な色味（#0a0a0f）を使う。セクション間で背景色をわずかに変える
8. 【見出し】一部分だけアクセントカラーやグラデーションにして抑揚をつける
9. 【装飾】どこか1箇所に「予想外」の要素を入れる（巨大な文字、はみ出し要素、斜めのdivider等）

━━━━━━━━━━━━━━━━━━━━
■ コード品質
━━━━━━━━━━━━━━━━━━━━

- 1つの .html ファイルに HTML + CSS + JavaScript を全て含める
- セマンティックHTML（<header>, <main>, <section>, <footer>）
- CSS変数でカラーパレットとスペーシングを管理
- モバイル対応（レスポンシブ）
- フォーカスリングを非表示にしない（アクセシビリティ）
- 画像は使わない（CSSのみでビジュアルを作る）
- コメントは最小限

━━━━━━━━━━━━━━━━━━━━
■ サイトのテーマ
━━━━━━━━━━━━━━━━━━━━

テーマ: [ここに自分のサイトのテーマを入れてください]
例: クリエイティブスタジオ / AIプロダクト / ポートフォリオ / テック系スタートアップ など

上記の全ルールを厳密に守り、「人間のトップデザイナーが丁寧に作り上げた」と思わせるクオリティで、完全なHTMLコードを出力してください。
```

## カスタマイズのヒント

**テーマ別に変えると良い部分：**

- **ポートフォリオ** → アクセントを #FFD60A（ゴールド）に、見出しフォントを 'Bebas Neue' に変更
- **SaaS / AI系** → 現状のままでOK。数字セクションにAPI呼び出し回数やユーザー数を入れる
- **クリエイティブ系** → アクセントを #FF006E メインに、Syne → 'Archivo Black' に変更
- **ラグジュアリー** → カラーを #C59D5C（ゴールド）に、フォントを 'Playfair Display' + 'Lato' に変更

**余白をさらに攻めたい場合：**
プロンプト内の `padding: 120px〜200px` を `padding: 200px〜320px` に変更してください。

**ライトモード版が欲しい場合：**
背景を `#FAFBFC`、テキストを `#1A1A1A`、サブテキストを `#666666` に差し替え、オーブの opacity を 0.05 程度に抑えてください。
