                          INSTALLATION COMMANDS
================================================================================

# Step 1: Install Charting & Icon Libraries
npm install recharts lucide-react

# Step 2: Install shadcn/ui Components
npx shadcn-ui@latest add card
npx shadcn-ui@latest add table
npx shadcn-ui@latest add button
npx shadcn-ui@latest add input
npx shadcn-ui@latest add dropdown-menu
npx shadcn-ui@latest add badge

# Step 3: Copy SaasDashboard.tsx to src/components/

# Step 4: Use in your page
import SaasDashboard from '@/components/SaasDashboard';

export default function Page() {
  return <SaasDashboard />;
}

# Step 5: Ensure tailwind.config.ts has:
darkMode: 'class'

# Done! 🎉

================================================================================
                            TROUBLESHOOTING
================================================================================

Dark mode not working?
  → Check tailwind.config.ts has: darkMode: 'class'

Charts not showing?
  → Run: npm install recharts
  → Check console for errors

Icons missing?
  → Run: npm install lucide-react

Styles broken?
  → Run: npm run build
  → Clear .next folder

Components not found?
  → Make sure shadcn/ui components are installed
  → Check component paths match your structure

More help?
  → Read SETUP_GUIDE.md (troubleshooting section)
