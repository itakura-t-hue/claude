# Landing Page Design Patterns

34 proven landing page patterns with design guidance, CTA placement strategies, conversion optimization, and implementation notes.

---

## 1. Classic Hero Section

| Aspect | Details |
|--------|---------|
| **Pattern Description** | Full-width hero with headline, subheading, CTA, and background image/video |
| **Hero Layout** | Full viewport height (100vh), background image with gradient overlay |
| **CTA Placement** | Center below subheading, 2-3 secondary CTAs optional |
| **Social Proof** | Logo list or client testimonial below CTA |
| **Conversion Optimization** | Keep headline under 10 words, benefit-focused CTA (not "Learn More") |
| **Color Strategy** | Dark overlay (40-60% opacity) on background image, high-contrast CTA |
| **Animation Recommendations** | Subtle fade-in on load, animate CTA button on scroll |
| **Mobile Adaptation** | Stack headline, reduce font size 30-50%, single CTA only |
| **Best For** | SaaS, tech startups, agencies |
| **Background** | High-quality image (1920x1080 min), video 5-10MB, autoplay muted |

## CTA Placement Best Practices

1. **Hero CTA**: Center-bottom or below headline, high contrast
2. **Secondary CTA**: Optional, lower prominence (e.g., "View Demo")
3. **Sticky CTA**: Fixed on scroll if hero CTA scrolls out of view
4. **Form CTA**: Button color matches form field border for cohesion
5. **Multiple CTAs**: Max 2-3, clear primary vs. secondary
6. **Mobile**: Larger touch target (48-56px height minimum)
7. **Button Text**: Action-oriented verbs ("Get Started", "View Pricing", not "Click Here")

---

## Conversion Optimization Checklist

- [ ] Headline is clear, benefit-driven, under 10 words
- [ ] Subheading expands on benefit, specific language
- [ ] CTA button color contrasts with background
- [ ] CTA text is action-verb based ("Get", "Start", "Join")
- [ ] Social proof visible (testimonials, logos, stats, trust badges)
- [ ] Visual supports message (no stock photos)
- [ ] Mobile responsive and readable
- [ ] Form (if any) has max 3 fields for hero section
- [ ] No more than 2-3 primary CTAs
- [ ] Above-the-fold content is fully visible (no scroll required on desktop)

---

## Mobile Optimization Guidelines

1. **Font Sizing**: Headline 28-36px, subheading 16-18px
2. **Touch Targets**: CTA buttons 48-56px height
3. **Spacing**: 16-24px margins, 8-16px padding
4. **Images**: Optimized for mobile (compressed, responsive)
5. **Form**: Single column, auto-focus first input
6. **Video**: Autoplay muted, provide play button
7. **Stack Order**: Content, then social proof, then CTA
8. **Scroll**: Hero height should fit on one screen without scroll

---

## A/B Testing Ideas

1. **CTA Text**: "Get Started" vs. "Start Free Trial"
2. **CTA Color**: Primary brand color vs. contrasting accent
3. **Headline**: Benefit-focused vs. feature-focused
4. **Social Proof**: Testimonial vs. logo wall vs. stats
5. **Hero Height**: Full viewport (100vh) vs. 70vh
6. **Form Fields**: Email only vs. Email + Name
7. **Image**: Photography vs. illustration vs. video
