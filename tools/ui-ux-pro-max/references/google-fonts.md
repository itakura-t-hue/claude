## Google Fonts Implementation

### Basic Import (CSS)
```css
@import url('https://fonts.googleapis.com/css2?family=Inter:wght@400;600;700&display=swap');

body {
  font-family: 'Inter', sans-serif;
}
```

### Optimized Link Tag
```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;600;700&display=swap" rel="stylesheet">
```

### React/Next.js Import
```typescript
import { Inter } from 'next/font/google';

const inter = Inter({ subsets: ['latin'] });

export default function RootLayout() {
  return (
    <html lang="en" className={inter.className}>
      ...
    </html>
  )
}
```

## Performance Tips

1. **Preconnect**: Add preconnect to fonts.googleapis.com and fonts.gstatic.com
2. **Weight Optimization**: Only load weights you use (400, 600, 700 usually sufficient)
3. **Subset**: Use `subsets=['latin']` for English-only sites
4. **Display**: Use `display=swap` for font loading strategy
5. **Variable Fonts**: Prefer variable fonts over multiple weights when available
6. **System Fallback**: Always provide system font fallback (sans-serif, serif, monospace)
