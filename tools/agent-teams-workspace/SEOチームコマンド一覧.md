# SEOディレクター補佐チーム コマンド一覧

## はじめに使うとき（初回セットアップ）

コマンドファイルは `~/.claude/commands/` に保存されています。
**Claude Codeの起動時に読み込まれる**ため、コマンドを追加・変更した場合は以下の手順でリロードしてください。

```
1. /exit と入力してClaude Codeを終了
2. ターミナルで claude と入力して再起動
3. /seo-team のようにコマンドを入力して使う
```

---

## コマンド一覧（早見表）

| コマンド | 人数 | ひとことで言うと | いつ使う？ |
|---------|------|----------------|-----------|
| `/seo-team` | 4名 | フル編成 | 大型案件、総合的なSEO支援 |
| `/seo-audit` | 2名 | SEO監査 | サイトの健康診断をしたいとき |
| `/seo-competitor` | 3名 | 競合分析 | 競合と自社を比較したいとき |
| `/seo-content` | 2名 | コンテンツ企画 | KW選定～記事構成案を作りたいとき |
| `/seo-report` | 2名 | 月次レポート | クライアント向けレポート作成 |
| `/seo-writing` | 2名 | 記事執筆 | 新規記事の執筆やリライト |
| `/seo-cro` | 2名 | CV改善 | CTA・バナー・LP・フォーム改善 |
| `/seo-architecture` | 2名 | サイト構造設計 | カテゴリ設計・内部リンク戦略 |
| `/seo-manage` | 2名 | 施策管理 | 施策の進捗管理・効果測定 |

## 各コマンドの詳細

### /seo-team（フル編成）

> 大型案件や総合的にSEOを見直したいときに使う、全部入りチーム

**メンバー構成**:

| メンバー | やること |
|---------|--------|
| 競合分析担当 | DR・トラフィック・被リンク分析、KW戦略分析 |
| テクニカルSEO担当 | サイト構造・内部リンク・構造化データ・Core Web Vitals |
| コンテンツ企画担当 | KWリサーチ・記事構成案・検索意図分析 |
| レポート担当 | 全員の結果を統合して総合レポート作成 |

**使い方**:
```
/seo-team
フォーティネットのSEO施策を総合的に見直したい。サイトはfortinet.com/jp
```

---

### /seo-audit（SEO監査）

> 特定サイトを技術面・コンテンツ面の両方からチェック

**メンバー構成**:

| メンバー | やること |
|---------|--------|
| テクニカル監査担当 | URL設計・内部リンク・構造化データ・robots.txt・sitemap・カノニカル・Core Web Vitals |
| コンテンツ監査担当 | タイトル・メタ・見出し構造・KW最適化・薄いコンテンツの特定 |

**成果物**: 優先度（高・中・低）付きの改善アクションアイテムリスト

**使い方**:
```
/seo-audit
https://www.example.com のSEO監査をお願い
```

---

### /seo-competitor（競合分析）

> 自社 vs 競合を数値とコンテンツの両面で比較して、施策ロードマップまで出す

**メンバー構成**:

| メンバー | やること |
|---------|--------|
| SEO指標分析担当 | DR・トラフィック推移・被リンク・KWランキング（Ahrefs） |
| コンテンツ戦略分析担当 | 競合のコンテンツ構成・上位記事の特徴・コンテンツギャップ |
| 施策提案担当 | 上記2名の結果を統合→差別化ポイント特定→施策ロードマップ作成 |

**成果物**: 比較表 + 短期（1-3ヶ月）・中期（3-6ヶ月）の施策ロードマップ

**使い方**:
```
/seo-competitor
自社: fortinet.com/jp
競合: paloaltonetworks.com, checkpoint.com
```

---

### /seo-content（コンテンツ企画）

> KW調査から記事構成案まで。クライアント提出可能な品質で作成

**メンバー構成**:

| メンバー | やること |
|---------|--------|
| キーワードリサーチャー | KW調査・検索ボリューム/KD確認・ロングテール発掘・検索意図分類 |
| コンテンツプランナー | KW選定結果→記事構成案・タイトル案・メタ案・E-E-A-T方針 |

**成果物**: KW選定シート + 記事構成案

**使い方**:
```
/seo-content
テーマ: 結婚指輪 買取
ターゲット: 結婚指輪を売りたい一般消費者
クライアント: K-GOLDインターナショナル
```

---

### /seo-report（月次レポート作成）

> Ahrefsでデータ収集→所感・考察付きの月次レポートを作成

**メンバー構成**:

| メンバー | やること |
|---------|--------|
| データアナリスト | トラフィック・KWランキング変動・被リンク増減・KPI集計（Ahrefs） |
| レポートライター | 数値をもとに所感・考察・次月施策提案を作成 |

**成果物**: データ集計 + 所感付き月次レポート（クライアント提出品質）

**使い方**:
```
/seo-report
クライアント: フォーティネット
対象サイト: fortinet.com/jp
対象月: 2026年2月
```

---

### /seo-writing（記事執筆・リライト）

> SEO記事を書く or リライトして、品質チェックまでやる

**メンバー構成**:

| メンバー | やること |
|---------|--------|
| SEOライター | 構成案に基づく執筆/リライト・KW配置最適化・E-E-A-T |
| 品質チェッカー | タイトル/メタ/見出し構造/KW密度/誤字脱字チェック→改善提案 |

**成果物**: 納品可能品質のSEO記事 + 品質チェックレポート

**使い方**:
```
/seo-writing
リライト対象: 【修正済】②【2026年版】刻印・傷があっても結婚指輪は売れる｜買取相場と安心して手放す方法.docx
主要KW: 結婚指輪 買取
```

---

### /seo-cro（コンバージョン改善）

> CTA・バナー・LP・フォームのCV改善施策を出す

**メンバー構成**:

| メンバー | やること |
|---------|--------|
| UX/導線分析担当 | ユーザー導線・CTA配置・LP構成・ファーストビュー・モバイルUX |
| コンバージョン改善担当 | CVR改善施策・CTAコピー案・EFO・A/Bテスト設計 |

**成果物**: 優先度付きCV改善施策リスト + CTAコピー案

**使い方**:
```
/seo-cro
対象: dym本体サイトのバナーとCTA
URL: https://dym.asia
```

---

### /seo-architecture（サイト構造設計）

> サイトのカテゴリ設計・URL設計・内部リンク戦略を立てる

**メンバー構成**:

| メンバー | やること |
|---------|--------|
| サイト構造設計担当 | ディレクトリ/URL設計・カテゴリ分類・パンくず・ピラーページ戦略 |
| 内部リンク設計担当 | 内部リンク構造設計・アンカーテキスト戦略・孤立ページ解消 |

**成果物**: サイト構造図（ツリー形式） + 内部リンク設計書

**使い方**:
```
/seo-architecture
対象: kin-kaitori.com
目的: 新規カテゴリ追加に伴う構造見直し
```

---

### /seo-manage（施策管理・効果検証）

> 施策管理表の更新と、実施済み施策の効果検証

**メンバー構成**:

| メンバー | やること |
|---------|--------|
| 施策管理担当 | 施策一覧の整理・進捗更新・優先度再評価・報告サマリー |
| 効果検証担当 | 施策の効果測定・KPI変動との紐づけ・ROI評価（Ahrefs） |

**成果物**: 施策管理表（更新済み） + 効果検証レポート

**使い方**:
```
/seo-manage
クライアント: ケアリッツアンドパートナーズ
施策管理表: 株式会社ケアリッツアンドパートナーズ_御中SEO施策管理表.xlsx
```

## チーム操作ガイド

| やりたいこと | 操作 |
|------------|------|
| チームメンバーを切り替える | **Shift+Down** |
| タスクリストを見る | **Ctrl+T** |
| メンバーのセッションを見る | **Enter** |
| 作業を中断する | **Escape** |
| チームを終了する | `Clean up the team` と入力 |

## 成果物の保存先

すべての成果物は `~/desktop/claude/agent-teams-workspace/` に自動保存されます。
