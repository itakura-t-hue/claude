# SEOディレクター補佐チーム ワークスペース

SEOディレクター業務を支援するAgent Teamsの作業スペース。

## 利用可能なコマンド

| コマンド | チーム構成 | 用途 |
|---------|----------|------|
| `/seo-team` | 4名（競合分析・テクニカル・コンテンツ・レポート） | フル編成。大型案件や総合的な支援に |
| `/seo-audit` | 2名（テクニカル監査・コンテンツ監査） | 特定サイトのSEO監査 |
| `/seo-competitor` | 3名（指標分析・コンテンツ戦略・施策提案） | 競合サイトの分析・比較 |
| `/seo-content` | 2名（KWリサーチ・コンテンツ企画） | KW選定と記事構成案の作成 |
| `/seo-report` | 2名（データ分析・所感作成） | 月次レポートの作成支援 |
| `/seo-writing` | 2名（SEOライター・品質チェッカー） | SEO記事の新規執筆やリライト |
| `/seo-cro` | 2名（UX導線分析・CV改善） | CTA・バナー・LP・フォームのCV改善 |
| `/seo-architecture` | 2名（サイト構造設計・内部リンク設計） | サイト構造・カテゴリ設計・内部リンク戦略 |
| `/seo-manage` | 2名（施策管理・効果検証） | SEO施策の進捗管理・効果測定 |

## 使い方の例

```
/seo-team
フォーティネットのSEO施策を総合的に見直したい。サイトはfortinet.com/jp
```

```
/seo-audit
https://example.com のSEO監査をお願い
```

```
/seo-competitor
自社: fortinet.com/jp
競合: paloaltonetworks.com, checkpoint.com
```

```
/seo-content
テーマ: 結婚指輪 買取
ターゲット: 結婚指輪を売りたい一般消費者
クライアント: K-GOLDインターナショナル
```

```
/seo-report
クライアント: フォーティネット
対象サイト: fortinet.com/jp
対象月: 2026年2月
```

```
/seo-writing
リライト対象: 【修正済】②【2026年版】刻印・傷があっても結婚指輪は売れる｜買取相場と安心して手放す方法.docx
主要KW: 結婚指輪 買取
```

```
/seo-cro
対象: dym本体サイトのバナーとCTA
URL: https://dym.asia
```

```
/seo-architecture
対象: kin-kaitori.com
目的: 新規カテゴリ追加に伴う構造見直し
```

```
/seo-manage
クライアント: ケアリッツアンドパートナーズ
施策管理表: 株式会社ケアリッツアンドパートナーズ_御中SEO施策管理表.xlsx
```

## 操作方法
- **Shift+Down**: チームメンバー間を切り替え
- **Ctrl+T**: タスクリスト表示
- チーム終了時: `Clean up the team`
