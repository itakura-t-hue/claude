## エグゼクティブサマリー

norst.co.jpの総合SEO監査を深掘りレベルで実施した。サイトはWordPress（カスタムテーマ）で構築され、全国36院の包茎手術専門クリニックサイトとして運営されている。

**最大の問題点は、ビッグキーワード「包茎手術」で検索上位10位圏外であること。** また「包茎手術 費用」「包茎手術 口コミ」「包茎手術 おすすめ」といった比較検討段階のキーワードでも軒並み圏外であり、CVに近いユーザーを競合に奪われている状態。

一方で、構造化データの実装は業界最充実であり、E-E-A-T（総院長の実名・経歴・著書の明記）も競合に対する明確な強みとなっている。テクニカルな修正とコンテンツ拡充を組み合わせることで、大幅な改善が見込める。

### 2-2. 新たに発見された重大な問題

#### (1) Productスキーマの価格有効期限が切れている
- **事実:** priceValidUntilが `2025-12-31` に設定されており、すでに期限切れ
- **影響:** Googleがリッチリザルト（価格表示）を非表示にする可能性
- **深刻度: S**

#### (2) MedicalClinicスキーマのopeningHoursが空値
- **事実:** 全院ページで openingHoursSpecification の opens/closes 値が空白
- **影響:** Googleマップ・ナレッジパネルで営業時間が正しく表示されない
- **深刻度: A**

#### (3) FAQのURL構造が二重化（カニバリゼーション）
- **事実:** 同カテゴリのFAQが2つのパスに存在
  - `/faq/phimosis/` と `/phimosis/faq/`
  - `/faq/powerup/` と `/powerup/faq/`
  - `/faq/trouble/` と `/trouble/faq/`
- **影響:** 同じ内容のページが複数存在し、Googleの評価が分散
- **深刻度: A**

#### (4) canonicalタグの設定が不完全
- **事実:** /phimosis/、/faq/、/reason/、/clinic/ 等にはcanonicalあり。**トップページ、/reserve/、/contact/ にはcanonicalなし**
- **修正:** 前回レポートの「全ページで未検出」は誤り。正しくは「一部ページで未設定」
- **深刻度: A**

#### (5) 実績数の表記がサイト内で不統一
- **事実:**
  - トップページ：「包茎手術の症例実績は**12万件以上**」
  - お役立ち情報記事（/content/phimosis/4128/）：「これまでの症例実績は**17万件**にのぼります（2025年1月時点）」
- **影響:** ユーザーに矛盾した情報を提供。信頼性の低下
- **深刻度: A**（YMYL領域で数値の不整合は致命的）

## 5. クリティカル改善施策（優先度順）

### 【S-1】包茎手術メインページの大幅リライト（最重要）
- **現状:** 5,000字以上、FAQ5項目。「包茎手術」で圏外
- **競合水準:** MSクリニック15,000-18,000字、FAQ15項目、Person構造化データ5名 → 2位
- **アクション:**
  1. コンテンツを15,000字以上に拡充
  2. ページ内FAQを15項目以上に増加 + FAQPageスキーマ実装
  3. 手術方法比較表（環状切除 vs 無切開 vs 修正再手術）を追加
  4. 包茎タイプ別図解を追加
  5. 年代別・タイプ別の患者統計データをグラフで可視化
  6. Person構造化データ（総院長）を実装
- **期待効果:** 「包茎手術」での上位表示獲得
- **工数:** 大 / **優先度: S**

### 【S-2】レピュテーション管理：口コミ・体験談ページの新設
- **現状:** 「東京ノーストクリニック 口コミ」で公式サイト非表示。第三者サイトにレピュテーションを完全委任
- **アクション:**
  1. 公式サイト内に「患者の声・体験談」ページを新設
  2. Review構造化データを実装
  3. 各院ページにも口コミセクションを追加
  4. 「東京ノーストクリニック 口コミ」でのSEO最適化
- **注意:** 医療広告ガイドラインに準拠した形式での掲載が必須
- **期待効果:** 指名検索＋口コミの検索結果をコントロール
- **工数:** 大 / **優先度: S**

### 【S-3】Productスキーマの価格有効期限を更新
- **現状:** priceValidUntil が `2025-12-31` で期限切れ
- **アクション:** `2027-12-31` に更新、または価格固定なら属性自体を削除
- **期待効果:** リッチリザルト（価格表示）の復活・維持
- **工数:** 小 / **優先度: S**（すぐできる）

### 【S-4】LINE公式アカウントの導入
- **現状:** 競合4社全て対応済み。norst.co.jpのみ未対応
