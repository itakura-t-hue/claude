## エグゼクティブサマリー

東京ノーストクリニック（norst.co.jp）の総合SEO監査を実施しました。サイトはWordPress（カスタムテーマ）で構築され、サイトマップ上に522URLが登録されています。基本的なオンページSEO（Title・H1・見出し構造・構造化データ）は概ね適切に実施されており、お役立ち情報コンテンツも180記事以上と充実しています。一方で、canonicalタグの未実装、OGPタグの欠如、サイトマップの長期未更新といったテクニカル面の課題が複数存在し、YMYL（医療）サイトとしてのE-E-A-T強化も急務です。競合分析では、LINE相談・動画コンテンツ・オンライン診療など、デジタル接点で競合に遅れを取っている点が明らかになりました。

### 最重要課題トップ5

| 順位 | 課題 | カテゴリ | 影響度 |
|------|------|----------|--------|
| 1 | canonicalタグが全ページで未実装 | テクニカル | 重複コンテンツリスク・クロール効率低下 |
| 2 | E-E-A-Tの「Experience（経験）」要素が弱い（症例写真・体験談の欠如） | コンテンツ | YMYL評価・競合との差別化に直結 |
| 3 | サイトマップが約4年半未更新（最終更新: 2021年11月） | テクニカル | 新規・更新ページのクロール通知不全 |
| 4 | FAQPageスキーマ未実装 | テクニカル | リッチリザルト獲得機会の大幅な逸失 |
| 5 | LINE相談未対応（競合4社中3社が対応済み） | 競合優位性 | 若年層ユーザーの取りこぼしリスク |

### 全体評価スコア

|----------|-------------------|----------|
| **テクニカルSEO** | 55点 | canonical・OGP未実装、サイトマップ未更新など基盤部分に課題が集中 |

### 3-4. norst.co.jpの競合優位性分析

#### 強み（活かすべきポイント）
1. **24時間営業** — 競合中唯一。深夜帯の相談需要を独占できるポジション
2. **料金の透明性** — 切らない手術33,000円、環状切除66,000円と明確公示
3. **院数の多さ** — 36院は業界第2位（メンズライフ39院に次ぐ）
4. **技術訴求** — マイクロスコープ使用、3段階麻酔、幹細胞培養上清液など差別化要素が豊富
5. **SEO基盤** — タイトルタグ・見出し構造は競合中最も最適化されている

#### 弱み（改善すべきポイント）
1. **LINE相談未対応** — 若年層の取りこぼしリスク
2. **知名度・ブランディング** — 上野クリニックはTV CM（ナダル起用）で圧倒的認知度
3. **オンライン診療非対応** — メンズライフが先行
4. **動画コンテンツ不足** — 上野クリニック・アトムクリニックに比べ動画活用が弱い
5. **実績数の見せ方** — 上野クリニック35万件に対して12万件以上

## 5. まとめ・次のステップ

### 現状の総括

norst.co.jp（東京ノーストクリニック）は、SEOの基盤（Title・見出し構造・構造化データの部分的実装）が一定水準で整っており、コンテンツ量も豊富です。24時間営業、料金透明性、全国36院展開といった事業上の強みも明確です。

しかし、テクニカルSEOの基本要素（canonical、OGP、サイトマップ）に課題が残っており、これらは比較的少ない工数で大きな改善効果が得られるため、最優先で対応すべきです。また、YMYL（医療）サイトとしてのE-E-A-T強化、特に「Experience（経験）」要素の改善は、Googleの評価基準とユーザーの信頼性の両面から中長期的な成長の鍵を握っています。

### 推奨する次のステップ

1. **即座に着手（今週中）**
   - SEOプラグイン（Yoast SEO / Rank Math）の導入検討・選定
   - canonicalタグ・OGPタグの一括実装
   - サイトマップの自動生成化

2. **1ヶ月以内に着手**
   - FAQPageスキーマの実装
   - 画像最適化（lazy loading、alt属性、WebP変換）
   - Meta Descriptionの個別設定
   - LINE相談導線の検討・導入

3. **3ヶ月以内に着手**
   - 症例写真・体験談ページの企画・制作
   - 「保険適用」専用LPの作成
   - 各院担当医師プロフィールの充実
   - 動画コンテンツの企画

4. **定期的に実施**
   - Google Search Console / PageSpeed Insightsでの定量データ計測（本監査ではAPIレート制限により未取得）
   - Core Web Vitalsの実測値に基づくパフォーマンス改善
   - 新規コンテンツの効果検証と追加施策の検討

### 補足事項

- 本レポートはWebFetchおよびWebSearchによるリモート分析に基づいています。PageSpeed InsightsのAPIレート制限により、Core Web Vitalsの定量データは取得できておりません。正確なパフォーマンスデータはGoogle Search ConsoleおよびPageSpeed Insightsでの直接計測を推奨します。
- 構造化データの検証はGoogle Rich Results Testでの確認を推奨します。
- 競合のSEO指標（DR、オーガニックトラフィック等）については、Ahrefs等のSEOツールでの詳細調査を別途実施することを推奨します。
