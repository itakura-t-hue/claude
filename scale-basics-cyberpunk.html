ROADMAP

学習ロードマップ

8つのフェーズで段階的にスキルを習得。各フェーズでは異なるWebサイトを0から構築・公開し、実践力を積み上げる。

Phase_01

Web入門

CH.01-04 // ポートフォリオサイト

セマンティックHTMLからTailwind CSSまで、Web制作の基本をマスター。Figmaでデザイン、Git/GitHubでバージョン管理し、GitHub Pagesで公開。

Phase_02

初級

CH.05-09 // コーポレートサイト

JavaScript/TypeScriptで動的ページを実装。サイト構造設計・SEOの基本を学び、Vercel公開。GA4導入でデータ計測の基礎を確立。

Phase_03

中級前半

CH.10-13 // ブログ/メディアサイト

WordPress×PHPでCMS構築。キーワードリサーチ・E-E-A-T対策・コンテンツSEOから被リンク戦略まで習得。

Phase_04

中級後半

CH.14-17 // SaaS型Webアプリ

React/Next.js+TypeScriptで動的Webアプリ構築。API連携・SSR/SSG/ISR・JavaScript SEOの課題を解決。

Phase_05

上級

CH.18-20 // 大規模EC/ポータル

DB設計・SQL・Shopify。大規模サイト特有のSEO課題（ファセットナビ、クロールバジェット、動的URL管理）を解決。

Phase_06

AI検索最適化

CH.21-23 // 既存サイト横断AIO

LLMO/GEO/AEOの実装。llms.txt設計・構造化データのAIO再設計。AI可視性を計測し統合戦略を構築。

Phase_07

データ分析深化

CH.24-26 // 多言語サイト+分析基盤

GA4×BigQuery連携でSQL分析。Looker Studioダッシュボード構築。Next.js i18nで多言語サイト公開。

Phase_08

総合実務

CH.27-29 // SEO監査・提案

テクニカル監査・コンテンツ監査・競合分析を一気通貫。改善提案書・月次レポート・プレゼンスキルを習得。
