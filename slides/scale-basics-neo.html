Phase 01

Phase 02

初級

Ch.05–09 / コーポレートサイト

JS/TS、サイト構造設計、テクニカルSEOの基本。Vercelに独自ドメインでデプロイして、GA4とGSCも導入。

Phase 03

中級前半

Ch.10–13 / ブログ・メディア

WordPress×PHP。キーワードリサーチ・E-E-A-T・コンテンツSEO・被リンク戦略を実践で学ぶ。

Phase 04

中級後半

Ch.14–17 / SaaS Webアプリ

React/Next.js+TypeScript。API連携・SSR/SSG/ISR・JavaScript SEO・CROまで。

Phase 05

上級

Ch.18–20 / 大規模EC・ポータル

DB/SQL/Shopify。ファセットナビゲーション、クロールバジェット最適化、EC特有のSEO。

Phase 06

AI検索最適化

Ch.21–23 / 横断AIO実装

LLMO/GEO/AEOの具体的な実装方法。llms.txt設計・構造化データ再設計・AI可視性の計測。

Phase 07

データ分析深化

Ch.24–26 / 多言語サイト + 分析

GA4×BigQuery、Looker Studio、Ahrefs/SEMrush。Next.js i18nで多言語サイトも構築。

Phase 08

総合実務

Ch.27–29 / SEO監査・提案

テクニカル監査・コンテンツ監査・競合分析。提案書の作り方、月次レポート、クライアント向けプレゼンテーション。
